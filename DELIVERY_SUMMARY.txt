================================================================================
TROUBLESHOOTING QUICK REFERENCE
================================================================================

Issue: Images not loading in browser
  Solution: Verify asset paths use /assets/agents/... (absolute from public/)
           Check NextJS public folder configuration
           Clear browser cache and hard refresh (Cmd+Shift+R)
           Verify WebP browser support

Issue: Colors look different than expected
  Solution: Check monitor color calibration
           Verify CSS filters aren't affecting colors
           Test in multiple browsers
           Check browser color management settings

Issue: File size or performance concerns
  Solution: Current WebP format is optimized for web
           Consider CDN caching for faster delivery
           Implement lazy loading in components
           Use responsive images with srcset if needed
