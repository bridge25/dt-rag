🔴 Case 1: STRAWBERRY의 R 개수 세기

❌ 문제 상황

프롬프트: "STRAWBERRY에서 R이 몇 개야?"

✅ 원인과 해결

해결: 문자열 분해 도구 사용

🔴 Case 2: Import 추측 오류

// AI가 추측으로 생성한 잘못된 import

// 실제 파일명은 userUtils.js였음

🔴 Case 3: "적절히 처리해줘" 지옥
