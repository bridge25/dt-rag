# SPEC-FRONTEND-REDESIGN-001: Acceptance Criteria

## Overview

이 문서는 Ethereal Glass & Constellation Explorer 프론트엔드 재설계의 완료 기준을 정의합니다.

---

## AC-001: 에이전트 카드 시스템

### 시각적 요구사항

| 항목 | 기준 | 검증 방법 |
|------|------|-----------|
| **디자인 일치도** | 뉴디자인1.png와 90%+ 일치 | 시각적 비교 검토 |
| **글래스 효과** | backdrop-blur + 반투명 배경 적용 | CSS 검사 |
| **카드 레이아웃** | 5열 그리드, 균일한 간격 | 레이아웃 검사 |
| **로봇 이미지** | 4종 희귀도별 커스텀 이미지 표시 | 이미지 로딩 확인 |

### 인터랙션 요구사항

| 항목 | 기준 | 검증 방법 |
|------|------|-----------|
| **호버 효과** | translateY(-4px) + 글로우 강화 | 마우스 호버 테스트 |
| **전환 부드러움** | 300ms ease 트랜지션 | 시각적 확인 |
| **shine 효과** | 호버 시 빛 스와이프 애니메이션 | 시각적 확인 |

### 희귀도별 스타일

| 희귀도 | 프로그레스 바 색상 | 글로우 색상 | 로봇 스타일 |
|--------|-------------------|-------------|-------------|
| Common | 회색 그라데이션 | 회색 (약함) | 기본 귀여운 로봇 |
| Rare | 청록색 그라데이션 | 청록색 | 발광 포인트 로봇 |
| Epic | 보라색 그라데이션 | 보라색 | 장식 있는 로봇 |
| Legendary | 금색 그라데이션 | 금색 | 왕관 장식 로봇 |

### 체크리스트

- [ ] 카드가 뉴디자인1과 시각적으로 일치함
- [ ] 16종 로봇 이미지 모두 로딩됨
- [ ] 희귀도별 색상 차별화 적용됨
- [ ] 호버 시 부드러운 상승 + 글로우
- [ ] 통계 데이터 (Users, Revenue, Growth) 표시
- [ ] Growth 값에 녹색 글로우 적용

## AC-002: Taxonomy Constellation 시스템

### 시각적 요구사항

| 항목 | 기준 | 검증 방법 |
|------|------|-----------|
| **디자인 일치도** | 뉴디자인2.png와 90%+ 일치 | 시각적 비교 검토 |
| **노드 형태** | 빛나는 유리 구체 스타일 | CSS/SVG 검사 |
| **연결선** | 부드러운 곡선 + 펄스 애니메이션 | 시각적 확인 |
| **배경** | 우주 테마 + 네뷸라 효과 | 배경 검사 |

### 인터랙션 요구사항

| 항목 | 기준 | 검증 방법 |
|------|------|-----------|
| **줌인/아웃** | 마우스 휠로 부드럽게 동작 | 휠 스크롤 테스트 |
| **패닝** | 드래그로 캔버스 이동 | 드래그 테스트 |
| **노드 호버** | 글로우 강화 + HOVERED 배지 | 마우스 호버 테스트 |
| **툴팁** | 호버 시 상세 정보 패널 표시 | 호버 테스트 |

### 컴포넌트별 체크

| 컴포넌트 | 요구사항 |
|----------|----------|
| **루트 노드** | 더 큰 크기, 강한 글로우, "ROOT NODE" 라벨 |
| **일반 노드** | 유리 구체, 아이콘, 하단 라벨 |
| **HOVERED 상태** | 청록색 테두리, "HOVERED" 배지, 상세 패널 |
| **연결선** | 흰색/청록색, 베지에 곡선, 펄스 애니메이션 |
| **컨트롤 패널** | 좌측 하단, 글래스 스타일, 4개 버튼 + 슬라이더 |

### 체크리스트

- [ ] 노드가 유리 구체 스타일로 렌더링됨
- [ ] 호버 시 HOVERED 배지 표시
- [ ] 호버 시 상세 정보 툴팁 표시
- [ ] 줌인/아웃 부드럽게 동작
- [ ] 패닝 부드럽게 동작
- [ ] 연결선에 펄스 애니메이션 적용
- [ ] 좌측 하단 컨트롤 패널 동작
- [ ] 루트 노드 특별 스타일링 적용
