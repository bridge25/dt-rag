PNG GENERATION
================================================================================

To generate PNG versions using Google's Gemini API:

1. Set API key:
   export GOOGLE_API_KEY="your-key-here"

2. Use prompts from NANOBANANA_PROMPTS.md:
   gemini "/generate '[robot-specific-prompt]'"

3. Move generated files:
   mv nanobanana-output/*.png apps/frontend/public/avatars/robots/

Alternatively, convert SVGs to PNG:
   convert -density 150 -background none *.svg *.png
